# Praneeth -- Developer Portfolio

A premium, interactive developer portfolio landing page showcasing Computer Science, Software Engineering, and AI/Deep Learning capabilities. Designed with modern aesthetics, dark theme, smooth motion graphics, and 3D card deck animations.

---

## 🌟 Key Features

* **3D CardStack Skills**: An interactive, fanned 3D stack showcasing skills grouped by categories (Languages, Frontend, AI/ML, Tools) with auto-play cycling.
* **Stacking Projects Deck**: Stacking portfolio cards that slide over each other on scroll. 
* **Interactive Project Galleries**: Clickable folders that pop with vibrant violet/magenta gradients and fan out to reveal resume points and project highlights.
* **Corner CSE & AI Ornaments**: Floating glassmorphic 3D SVGs of a neural network brain, code brackets, silicon processor, and server databases.
* **Services Menu**: Spring hover magnification that highlights capabilities in Frontend, AI, UI/UX, Data, and Software Engineering.
* **Dotted Wave Surface**: A smooth particle wave background rendered dynamically.

---

## 🛠️ Technology Stack

* **Core**: React 18, TypeScript, HTML5, CSS3 (Vanilla CSS)
* **Styling**: Tailwind CSS
* **Animations**: Framer Motion
* **Icons**: Lucide React
* **Build Tool**: Vite

## 🚀 Setup & Execution

### Prerequisites
Make sure you have Node.js installed on your computer.

### Installation
1. Clone this repository (or navigate to your directory).
2. Install the necessary dependencies:
   ```bash
   npm install
   ```

### Running Locally
To launch the hot-reloading development server:
```bash
npm run dev
```
Open **[http://localhost:5173](http://localhost:5173)** in your browser.

### Building for Production
To bundle and optimize the project for deployment:
```bash
npm run build
```
The production bundle will be generated in the `/dist` directory.
